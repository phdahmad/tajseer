## الميزات

### للطلبة
يدخل الطالب برقمه الجامعي ويشاهد:
- الدرجة النهائية والنسبة المئوية مع رسم دائري متحرك
- التقدير (ممتاز / جيد جداً / جيد / مقبول / راسب)
- توزيع الدرجة على اقسام الاختبار
- التصحيح التفصيلي لكل سؤال مع الاجابة الصحيحة والملاحظة
- ملاحظات وتوصيات خاصة بالطالب
- تحميل التقرير الكامل بصيغة PDF او Word
- تحميل ورقة الاجابة الاصلية المسحوبة ضوئياً

### للمدرس (لوحة التحكم)
بعد الدخول بكلمة السر `تجسير` يشاهد المدرس:

**1. ملخص احصائي**
- عدد الطلبة، متوسط الدرجات، اعلى/ادنى درجة
- عدد الناجحين والراسبين

**2. توزيع التقديرات**
رسم بياني افقي يعرض عدد الطلبة في كل تقدير

**3. قائمة الطلبة وتقاريرهم** (الاهم)
- جدول مرتب من الاعلى للادنى مع ميداليات للمراكز الاولى
- بحث وفلترة بالتقدير وترتيب بالاسم/الرقم/الدرجة
- زر عرض تفاصيل لكل طالب (نافذة كاملة بالتصحيح)
- ازرار تحميل لكل ملف منفرد (PDF / Word / ورقة الاجابة)
- زر تحميل **كل التقارير دفعة واحدة** في ملف ZIP

**4. نشاط الدخول** (في اسفل الصفحة)
- بطاقات احصائية: اجمالي التسجيلات، عدد من دخل، عدد من لم يدخل، المحاولات الفاشلة
- رسم بياني للدخول حسب اليوم
- جدول نشاط كل طالب (عدد المرات، اول/اخر دخول، الجهاز والمتصفح)
- قائمة الطلبة الذين لم يدخلوا (للتذكير)
- محاولات الدخول الفاشلة (لاكتشاف الفضوليين)
- سجل كامل بكل التسجيلات مرتبة من الاحدث

- نوع المتصفح (Chrome/Safari/Firefox/Edge)

## كيفية تحديث الموقع

### تحديث المحتوى (HTML/CSS/JS)
عدّل الملفات محلياً ثم ارفعها لـ GitHub:
1. ادخل https://github.com/phdahmad/tajseer
2. اختر الملف ← اضغط القلم ✏ للتعديل المباشر، او ارفع نسخة جديدة
3. Commit changes ← انتظر دقيقة لين GitHub Pages يحدث

### تحديث كلمة سر لوحة التحكم

- `admin.html`: السطر `const PASSWORD = "تجسير";`

## استكشاف الاخطاء

### "Failed to fetch" في لوحة التحكم
- تأكد ان Apps Script منشور بصلاحية "أي شخص"
- جرب فتح الرابط مع callback في المتصفح:
  ```
  https://script.google.com/macros/s/.../exec?callback=test&password=تجسير
  ```
  يجب ان ترى: `test({"ok":true,"rows":[...]})`
- اذا لم يعمل، اعد نشر Apps Script (New version)

### الطلبة لا تظهر تسجيلاتهم
- ادخل Apps Script → Executions
- يجب ان ترى doPost executions باللون الاخضر "مكتملة"
- اذا فيه فشل، اضغط على execution وشوف الخطأ

### "Cannot read properties of null"
السكربت يحاول الوصول لشيت غير موجود او بدون صلاحية. الحل:
1. شغل دالة `getOrCreateSheet` يدوياً من Editor
2. اقبل صلاحية الوصول لـ Sheets
3. اعد النشر
